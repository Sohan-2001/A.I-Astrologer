# **App Name**: Chirp Chat

## Core Features:

- Message Input: Text input field for composing messages.
- Send Message: Button to send the message.
- Customizable Chat Bubbles: Visually distinct chat bubbles for sent and received messages.
- Scrollable Conversation History: Display of past messages with smooth scrolling.
- Contact Header: Display the contact name or group name at the top of the chat.

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB) for a friendly and approachable feel.
- Background color: Off-white (#F0F8FF) to provide a clean and modern canvas.
- Accent color: Soft Blue (#ADD8E6) for subtle highlights and interactive elements.
- Body and headline font: 'PT Sans' sans-serif for a modern and readable design.
- Simple, outlined icons for a clean interface.
- Modern layout, with fixed header and message input, and scrollable messages in between.
- Smooth transitions for sending/receiving message animations.
